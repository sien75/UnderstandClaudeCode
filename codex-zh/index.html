OpenAI Codex

这是 Codex 开源单仓库的代码理解地图。项目以 Rust 为主体，提供 CLI、终端交互界面与 App Server；TypeScript 和 Python SDK 将这些能力嵌入其他应用。核心工作是把用户目标变成带上下文的模型请求，在审批与沙箱约束下调用本地或远程工具，并持续保存线程历史、状态和事件。

用户和外部系统可以做什么

  • 从 CLI 输入任务、恢复会话并运行完整代理回合 → 用户从 CLI 发起并完成一次代理回合
  • 在终端中编辑输入、查看流式输出并响应审批 → 终端交互
  • 从桌面端或 IDE 通过 RPC 创建线程、启动/引导/中断回合 → App Server RPC
  • 通过 TypeScript 或 Python SDK 嵌入、流式消费和控制 Codex → SDK 集成与运行时
  • 选择配置、登录方式和模型供应商 → 配置、认证与模型连接
  • 让代理执行命令、应用补丁、调用 MCP，并接受审批与沙箱保护 → 工具执行、审批、沙箱与 MCP
  • 保存、恢复、压缩和迁移线程历史 → 核心状态与历史
  • 安装插件、技能与扩展，连接云任务、网络代理和遥测 → 支撑服务
